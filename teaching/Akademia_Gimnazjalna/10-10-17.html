Zajęcia 10 października

Omówione zagadnienia

  • Przypomnienie pojęcia zmiennej.
  • Typy danych: int, long long int, float, double, char, bool.
  • Niedokładność typów float i double.
  • Wypisywanie liczb zmiennoprzecinkowych z ustaloną precyzją (za pomocą cout) - funkcje fixed oraz setprecision.
  • Powtórzenie instrukcji warunkowej.

Materiały

  • Wczytywanie i wypisywanie liczby zmiennoprzecinkowej z ustaloną dokładnością
    #include<iostream>
using namespace std;
int main(){
    double x;
    cin >> x;
    cout << fixed << setprecision(6) << x << "\n";
    return 0;
